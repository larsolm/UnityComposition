Instruction Graph

An abstract Asset that is the main component of the composition system. Create graphs and use the Instruction Graph Window to build Nodes that execute actions, and create connections to traverse through each node sequentially, similar to a flow chart. An Instruction Graph can have Input Values that are passed in from the calling object and Output Values that it returns to the calling object. All graphs have a Context Value which is usually set to the object that originally ran the graph. Edit the Definitions for the Context, Inputs, and Outputs in the inspector for an Instruction Graph, and each of them will automatically be cast to the correct type in script. Built in Instruction Graph types are the Simple Graph, Scoped Graph, and the Mockup Graph (used for prototyping).

See Graphs for more information.

Name Description

Context Name

The string name of the context Value to be accessed in Expressions and VariableReferences
