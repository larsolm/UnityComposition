Collision Graph Trigger

Collision Graph Trigger is an Instruction Trigger that will run an Instruction Graph on collisions. Collision Graph Trigger works with all (2D and 3D) of Unity’s enter and exit collision messages which will run the respective EnterGraph and ExitGraph when they are called.

See Graphs for more information on instruction graphs.
See the "Maze1" scene in the Maze project for an example usage.

Fields

Name Description

Enter Graph

The Instruction Graph to run when this object is collided with.

Exit Graph

The Instruction Graph to run when this object stops colliding.
