Nodes

Nodes make up the bulk of the functionality of a graph. They control both the flow of the graph and the behavior that should be performed. For example, a Conditional Node branches the graph based on the result of an expression and has no effect on the game world, while a Message Node physically displays a message in the game. The following is a list of built-in nodes by category:

Animation

Play Animation

Play an animation on a game object

Play Animation State

Play an animation state on an Animator

Set Animation Parameter

Set an animation parameter on an Animator

Play Effect

Create and play an effect Prefab

Play Sound

Play an AudioClip

Play Timeline

Run a timeline
Composition

Expression

Run an expression

Instruction

Run an instruction

Shuffle

Shuffle a variable list

Sort

Sort a variable list

Reset Tag

Reset variables by tag

Reset Variables

Reset variables by name
Control Flow

Conditional

Fork the graph based on a condition

Branch

Run a single branch based on a string value

Sequence

Run a list of nodes in order

Loop

Loop until a condition is met

Iterate

Iterate the items in a variable list

Break

Break out of a loop

Yield

Yield for a single frame

Yield

Break out of all loops
Debug

Comment

Keep notes in the graph

Log

Log a message in the console

Mockup

For visual prototyping
Interface

Show Message

Show a message

Show Selection

Show and wait for a selection

Input

Wait until a button is pressed

Show Control

Activate an interface control

Hide Control

Hide an interface control

Set Binding

Object Manipulation

Create Game Object

Sequencing

Play Transition

Load Scene

Play a transition

Play a transition

Transform Object

Time Scale

Set the time scale value

Wait
