Getting Started:
----------------
o To run the examples you MUST add all scenes to the build settings. To do this automatically, click on the menu item, "Tools/Add Examples to Build."
o Open the "Loader" scene located in the "Assets/Composition Framework/Loader" folder
o When prompted, choose to import the default TextMesh Pro resources
o Play the Loader scene and choose which example you want to run
o Examples can be played using a mouse, controller, or keyboard
o To view the documentation open the "Documentation.html" file located in "Assets/Composition Framework/Documentation" folder in a web browser.
o Visit "https://pirhosoft.com/projects/unity-composition/documentation/#/v10/" for the most up to date, online documentation and tutorial videos.
o To use the framework on its own simply delete the Examples folder or choose to not import it initially.
