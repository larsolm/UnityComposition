Variable Binding

A Variable Binding is an abstract MonoBehaviour used to bind data from the composition system, usually via VariableReferences, to visual elements in the scene. By default a Variable Binding has access to the Composition Manager's "global" and "scene" IVariableStores. If a Variable Binding has a parent or parents with a Binding Root they can access the Value property on those Binding Roots via their ValueName property. Variable Bindings can be categorized into groups with the BindingGroup property so that only certain bindings will update when they are prompted (usually by an Update Binding Node. If AutoUpdate is set then Variable Bindings will update automatically every frame. Most Variable Bindings will disable their corresponding visual element if they fail to retreive their data and report the error, however, sometimes this may be intended behaviour so if set, SuppressErrors will hide those errors. Some example implementations of a Variable Binding are Enable Binding, Image Binding, and String Binding.

See Variable Bindings for more information.

Fields

Name Description

Binding Group

The string name of the group this binding belongs to

Auto Update

Whether to automatically update this binding every frame

Suppress Errors
